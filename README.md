# PRO1-FIB

Coursework repository for **PRO1 (Programación I)** at the **Facultat d'Informàtica de Barcelona (FIB - UPC)**.

This repository gathers introductory programming exercises and Jutge problem sets completed during the course.

## Overview

PRO1 is the starting point for programming at FIB and focuses on fundamental problem solving, control flow, data handling, and structured thinking.

This repo serves as a clean record of early programming practice and coursework progress.

## Contents

- Jutge problems
- Introductory programming exercises
- Practice material
- Basic problem-solving code

## Tech

- C++
- Programming fundamentals
- Structured problem solving

## Notes

This is a foundational academic repository for the PRO1 coursework at FIB.

It reflects early-stage programming development and is kept mainly as a documented archive of that stage.
